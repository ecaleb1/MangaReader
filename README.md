# MangaReader

## A Simple comic/manga reader written in Rust using Iced

### Usage:
`MangaReader [file]`

### Controls:
`UP`            Move view up
`DOWN`          Move view down
`SHIFT+UP`      Increase zoom
`SHIFT+DOWN`    Decrease zoom
`HOME`          Jump to first page
`END`           Jump to last page
`ESC/Q`         Exit the program
